# sap_ai_video

这个项目用于沉淀和创作微信公众号内容。

当前内容方向聚焦在 **AI + SAP / SAP 乙方交付**：

- 用 SAP 乙方团队熟悉的交付语言，讲清楚 AI 如何进入 SAP 项目与运维交付流程。
- 将公众号选题、文章框架、表达口径、案例素材和交付方法论统一沉淀到文档中。
- 后续可围绕母文档拆分成系列文章、视频脚本、公众号推文和销售沟通材料。

## 目录结构

- `doc/资料库文件/`：按场景沉淀素材、案例、术语和交付口径。
- `doc/management_docs/`：账号定位、母文档、栏目规划和内部文章状态管理表。
- `doc/column_content/`：按栏目和文章归档正式内容，文章内部拆分为 `markdown/`、`html/`、`assets/` 和 `source/`。
- `doc/photo/`：创作过程中的图片源文件和可复用视觉素材。
- `old/`：已经合并、迁移或归档的旧版本文件。
